Introduction

This package contains a LaTeX class file and a template for a UCSD Ph. D. thesis.

For instructions see the readme or the wiki pages:
(currently hosted at https://code.google.com/p/ucsd-thesis)

  • [GettingStarted Getting Started]
  • [TroubleShooting Troubleshooting]

To generate the final PDF, it is recommended that the pdflatex command is used. If you require Unicode another method may be better suited, see the [AdvancedUsage Advanced Usage] wiki page.


For questions and comments go to the ucsd-thesis Google group.

This project is maintained by UCSD graduate students. Let us know when you submit a thesis using this package.

If you run into problems, let us know by using the issue tracker.

If you need to contact the developers directly add @gmail.com after our username.

The class file in this project was originally adapted from:

  • http://sourceforge.net/projects/ucsd-thesis/
  • http://math.ucsd.edu/~thesis/resources.html

This package has been worked on by many many graduate students over the years. A partial list of authors can be found in the AUTHORS file.
